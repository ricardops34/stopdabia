> Crochet amigurumi icon, thick fluffy yarn texture, soft 3D look, handmade cozy aesthetic, vibrant saturated colors, isolated on transparent background, no text, no letters, same art style as the reference image provided

Cole este prompt antes de qualquer prompt específico abaixo.

## 5. Dicas para geração

- **Midjourney:** use `--style raw --v 6` e anexe as imagens de referência com `--sref`
- **DALL-E 3:** descreva "same style as [referência]" e cole o prompt
- **Stable Diffusion:** use as imagens de referência como IP-Adapter
- **Adobe Firefly:** "Generative Fill" com estilo crochê

### Verificação de consistência
Antes de aprovar uma imagem, cheque:
- [ ] Fundo transparente (se PNG)
- [ ] Dimensões corretas
- [ ] Estilo crochê 3D consistente com os avatares
- [ ] Sem texto ou letras na imagem
- [ ] Cores vibrantes e saturadas
